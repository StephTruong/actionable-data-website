Project Job Fiction

A new way to find your ideal job

Focus

The idea of this project spark out of an overall feeling as Data Scientist on the lookout for a suitable position. In the current situation the job title of Data Scientist can mean anything ranging from Data Analyst to Senior Director of Analytics. To fix this pain point we have released a search engine that extracts and matches keywords instead of a mere job title. Our product lets the user easily submit the most relevant keywords and discover job ads that matches those keywords. The results are sometimes surprising as we are to able discover suitable job positions with high ranked job title.

Take Control

To achieve this we spin out a Linux server with MongoDB. The job extraction task is written in Python with the help of the Indeed API. Extracting relevant keyword is not an easy feat but by leveraging MAUI Indexer, a NLP keyword extraction tool, we obtain highly satisfying results.

Discover

The recommender systems we use to feed results is based on a Latent Dirichlet Allocation algorithm with the GenSim library. By classifying jobs from our database into several topics and by passing through multiple Expectation Maximization pass, the system is able to recommend jobs with great accuracy.

Enlighten

The end product is a fully functional website build on Flask back end and a Bootstrap/HTML5 front end. We use D3.js to build some visualizations that makes the results more compelling.
